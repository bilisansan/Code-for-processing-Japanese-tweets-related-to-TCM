# Code-for-processing-Japanese-tweets-related-to-TCM

This project analyzes large-scale Japanese tweets related to Traditional Chinese Medicine (TCM) using BERTopic and multilingual sentiment analysis.

## Pipeline

1. Extract tweets
2. Preprocess 
3. Tokenization
4. Sentence embedding
5. Topic modeling
6. Sentiment analysis

## Directory Structure

- config/ configuration files
- src/ source files
- src/ main scripts


## Run Example

-- python src/extract_data.py
- python src/preprocess.py
- python src/emb_sentence.py
- python src/main.py
- python src/sent_analysis.py
- python src/README.md
